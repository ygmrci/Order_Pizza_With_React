## Proje Özeti

Bu proje, Workintech Full-Stack programı kapsamında geliştirilen
**Teknolojik Yemekler** adlı bir Single Page Application’dır.

Uygulamada kullanıcı:

- Anasayfadan ürün seçebilir
- Sipariş formunu doldurabilir
- Form validasyonları tamamlanmadan sipariş veremez
- Siparişi gönderdiğinde mock bir API’ye POST isteği atılır
- Sipariş başarıyla tamamlandığında onay ekranına yönlendirilir

Proje, React Router, controlled form yapısı, prop-lifting ile state yönetimi
ve Cypress E2E testleri kullanılarak geliştirilmiştir.

---

## Kullanılan Teknolojiler

- React
- React Router DOM
- Axios
- CSS (Flexbox / Grid)
- Cypress (E2E Testing)
- Reqres.in (Mock API)

---
